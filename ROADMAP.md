# Roadmap

## Short term

- Improve per-command safety controls (`/shell`, `/docker`).
- Add a read-only mode for diagnostic queries.
- Publish a hardening guide for internet-exposed deployments.

## Mid term

- Unit test suite for core functions in `bot.py`.
- Integrate linter/formatter in CI.
- Issue/PR templates for open collaboration.

## Long term

- Optional support for Telegram webhooks (in addition to polling).
- Basic observability (metrics and bridge health alerts).
